# Bone Fracture Image Classifier

## Image Processing Module

Put X-ray images into `/data/input` folder.

Output should generate augmented images in the `/data/output` folder (ignore the blank file in `output` folder, its just so i can add a blank folder).

## Run Tests

Put dataset into '/data/' folder
Install dependencies (pip install -r requirements.txt)

Then run:

```
python example.py
```

from the project root.
